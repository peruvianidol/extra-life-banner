Extra Life Banner 2.0

This is a 1920 x 1080 page that you can pull into OBS as an overlay to display your Extra Life goals and donors in (almost) real-time. At the top of the JS, you can set the following values to customize the data:

participantIdYou can find this in the URL for your donation page
teamIdYou can find this in the URL of your team donation page
donationUrlThis is the URL that will appear in the overlay
socialNameYour username on Twitter/Twitch
intervalHow often in milliseconds you want the data to refresh (default: 60s)

You can also customize the colors and font used in the overlay using CSS custom properties. You'll need to change the Google font used in the <head> in order to update the --font-family custom property in the CSS.
